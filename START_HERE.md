## ✨ 核心功能

### 🎬 视频模块
- 搜索视频（带历史记录）
- 分类筛选（电影/电视剧/综艺/动漫/纪录片/短剧）
- 视频详情和播放
- 多播放源支持
- 收藏和历史记录
- 40+ 视频站点

### 🖼️ 图片模块
- 随机图片浏览
- 同源/换源刷新
- 自动刷新（1/3/5秒）
- 图片缩放
- 20+ 图片站点

### 🎥 短视频模块
- 短视频播放
- 换一个/换分类
- 自动循环
- 20+ 短视频站点

### 💾 用户数据
- 搜索历史（10条）
- 播放历史（50条）
- 收藏管理
- 数据清理

## 🎯 使用流程

### 1. 安装 Flutter
如果还没安装，查看 [doc/INSTALL.md](doc/INSTALL.md)

### 2. 安装依赖
```bash
cd D:\project\dart\MeloBoxApp
flutter pub get
```

### 3. 运行应用
```bash
# 方式1
run.bat

# 方式2
flutter run
```

### 4. 构建 APK
详见 [doc/BUILD_GUIDE.md](doc/BUILD_GUIDE.md)
```bash
flutter build apk --release --android-skip-build-dependency-validation
```

## 🎨 功能展示

### 主界面
```
┌────────────────────────────┐
│ ☰ MeloBox  ⋮ ♥ ⏱ ↻       │
├────────────────────────────┤
│ 🔍 搜索视频...        🔍   │
│ 📺 当前站点               │
│ [全部][电影][电视剧]...   │
├────────────────────────────┤
│ 视频网格                   │
└────────────────────────────┘
```

### 功能入口
- **☰** 侧边栏 - 80+ 站点切换
- **⋮** 更多 - 图片/短视频/下载
- **♥** 收藏 - 收藏列表
- **⏱** 历史 - 播放历史
- **↻** 刷新 - 刷新列表
