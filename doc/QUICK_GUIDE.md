# MeloBox App - 快速参考指南

## 📱 功能入口

### 主页面
```
┌─────────────────────────────────┐
│  ☰  MeloBox      ⋮ ♥ ⏱ ↻       │  
├─────────────────────────────────┤
│  🔍 搜索视频...            🔍   │
│  📺 当前站点: xxx               │
├─────────────────────────────────┤
│  [视频网格列表]                 │
│  ┌────┬────┐                    │
│  │ 📺 │ 📺 │                    │
│  └────┴────┘                    │
└─────────────────────────────────┘
```

### 功能按钮说明
- **☰** - 侧边栏（站点切换）
- **⋮** - 更多菜单（图片/短视频）
- **♥** - 收藏列表
- **⏱** - 播放历史
- **↻** - 刷新列表

---

## 🎬 视频功能

### 搜索视频
1. 点击搜索框
2. 输入关键词
3. 按回车或点击搜索按钮
4. 查看搜索历史（点击快速填充）

### 浏览视频
- 网格布局展示
- 封面图片
- 标题、类型、状态
- 点击卡片查看详情

### 观看视频
1. 点击视频卡片 → 详情页
2. 选择播放源（如有多个）
3. 点击集数按钮开始播放
4. 自动记录播放历史

### 收藏视频
1. 详情页右上角点击 ♥
2. 再次点击取消收藏
3. 主页点击 ♥ 查看收藏列表

---

## 🖼️ 图片功能

### 打开图片浏览器
主页 → 右上角 ⋮ → 随机图片

### 操作说明
```
┌─────────────────────────────────┐
│  ←  随机图片          ⏱ 3s     │
├─────────────────────────────────┤
│  [同源刷新] [换源刷新] [自动]    │
│  当前: ACG图片                   │
├─────────────────────────────────┤
│                                 │
│         [图片显示区域]           │
│      （可双指缩放）              │
│                                 │
└─────────────────────────────────┘
```

### 刷新模式
- **同源刷新** - 当前站点换一张
- **换源刷新** - 切换到下一个图片站
- **自动** - 自动循环刷新

### 刷新间隔
点击右上角 ⏱ 选择：
- 1秒
- 3秒（默认）
- 5秒

### 打开短视频
主页 → 右上角 ⋮ → 短视频

```
┌─────────────────────────────────┐
│  ←  短视频                      │
├─────────────────────────────────┤

│       [视频播放区域]             │
```

## 🔍 搜索技巧

### 关键词建议
- 电影名称：例如 "复仇者联盟"
- 演员名称：例如 "成龙"
- 类型：例如 "动作"
- 年份：例如 "2024"

### 搜索历史
- 自动保存最近 10 个搜索
- 点击历史快速搜索
- 点击 X 删除单条
- 设置中清空全部

## ⚠️ 常见问题

### 视频无法播放
1. 检查网络连接
2. 尝试其他播放源
3. 切换到其他站点
4. 检查视频地址是否有效

### 图片加载失败
1. 点击"同源刷新"重试
2. 点击"换源刷新"换站点
3. 检查网络连接

### 短视频无法加载
1. 点击"换一个"重试
2. 点击"换分类"换站点
3. 检查网络连接

### 搜索无结果
1. 尝试其他关键词
2. 切换到其他站点
3. 清空搜索查看全部列表
