## 📡 API 接口

### 视频接口
```bash
# 搜索
GET {baseUrl}?ac=list&wd={keyword}&pg={page}

# 列表
GET {baseUrl}?ac=list&pg={page}

# 分类
GET {baseUrl}?ac=list&t={categoryId}&pg={page}

# 详情
GET {baseUrl}?ac=detail&ids={videoId}
```

### 媒体接口
```bash
# 图片
GET {baseUrl}?t={timestamp}

# 短视频
GET {baseUrl}&t={timestamp}
```

## 💡 使用建议

### 首次使用
1. 安装 Flutter SDK
2. 运行 `flutter pub get`
3. 连接设备或模拟器
4. 运行 `flutter run`

### 构建 APK
```bash
# 方式1: 使用脚本
.\build_apk.bat

# 方式2: 命令行
flutter build apk --release
```

### 日常开发
```bash
# 开发模式
flutter run

# 热重载
按 r 键

# 热重启
```
