## 📱 使用流程

### 视频观看流程
```
1. 打开应用
   ↓
2. 搜索或选择分类 🆕
   ↓
3. 浏览视频列表
   ↓
4. 点击视频卡片
   ↓
5. 查看详情，点击收藏 ♥
   ↓
6. 选择播放源和集数
   ↓
7. 开始播放（自动记录历史）
```

### 图片浏览流程
```
主页 → ⋮ 更多 → 随机图片 → 刷新模式选择
```

### 短视频流程
```
主页 → ⋮ 更多 → 短视频 → 切换视频
```

## 🔧 技术实现

### 架构设计
```
┌─────────────────────────────┐
│     Presentation Layer      │
│  (Screens + Widgets)        │
├─────────────────────────────┤
│   Business Logic Layer      │
│  (Providers - 7个)          │
├─────────────────────────────┤
│        Data Layer           │
│  (Models + Network + Cache) │
└─────────────────────────────┘
```

### 状态管理
```
Provider Pattern
├─ SiteProvider (站点)
├─ VideoProvider (视频)
├─ SearchHistoryProvider (搜索历史)
├─ PlayHistoryProvider (播放历史)
├─ FavoriteProvider (收藏)
├─ VideoCategoryProvider (分类) 🆕
└─ DownloadProvider (下载) 🆕
```
